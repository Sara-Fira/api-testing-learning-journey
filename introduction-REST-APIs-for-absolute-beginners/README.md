# Introduction to REST APIs for absolute beginners — Postman Learning Collection (Udemy Guided Course)

This repository contains my Postman practice collection created while learning REST API fundamentals.
The practice was performed mainly on Car Fleet Management API.

## What I practiced

- HTTP request and response structure
- GET, POST, PUT and DELETE requests
- Query parameters and path parameters
- Request headers
- JSON request body
- HTTP status codes
- Reading API responses in Postman
- Basic REST API workflow

## Tools used

- Postman
- REST API practice endpoints
- GitHub
- Udemy

## Purpose

The purpose of this repository is to document my learning progress in REST API basics and Postman as part of my QA learning path.
