# Postman Academy API Beginner Path

This folder contains my guided practice from the **Postman Academy API Beginner Path**.

The path introduced the basics of APIs, the Postman platform, and making first API requests in Postman.

## Topics Practiced

* API fundamentals
* Postman basics
* REST API requests
* responses
* endpoints
* HTTP status codes
* JSON basics
* organizing requests in a Postman collection

## Tools Used

* Postman
* REST API
* JSON

## Files

- `collections/` — exported Postman collection from guided practice
- `environments/` — exported Postman environment used during practice, without sensitive data

## Learning Context

This is a beginner-level guided practice folder created as part of my QA learning journey. It documents my first practical exposure to API requests and basic Postman workflows.
